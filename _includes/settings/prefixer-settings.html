デフォルトでは、 Bourbon はそれぞれのミックスインに明記された全てのベンダープレフィックスを出力します。このグローバルなデフォルトは、あなたのスタイルシートの先頭で、各変数に false を設定することで上書きすることもできます。

{% highlight scss %} $prefix-for-webkit: true; $prefix-for-mozilla: true; $prefix-for-microsoft: true; $prefix-for-opera: true; $prefix-for-spec: true; {% endhighlight %}
