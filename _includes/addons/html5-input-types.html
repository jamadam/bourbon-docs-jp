HTML5 Input Types

ソースを見る

このアドオンは、HTML5のinputタイプのリストを格納したいくつかの変数を提供します。This addon provides you with five variables which each contain a list of all html5 input types.

このうち3つは、textareaを除いたテキストベースのinputを出力します。言い換えると、これらは input[type="text"] を真似て、すべてのinputを簡単に指定することを可能にします。

  • #{$all-text-inputs}
  • #{$all-text-inputs-hover}— all-text-inputs を指定するセレクターに :hover 疑似クラスを付与して出力します
  • #{$all-text-inputs-focus}— all-text-inputs を指定するセレクターに :focus 疑似クラスを付与して出力します
  • #{$all-text-inputs-active}— all-text-inputs を指定するセレクターに :active 疑似クラスを付与して出力します

このうち4つは、buttonを除いたボタンベースのinputを出力します。言い換えると、これらは input[type="submit"] を真似て、すべてのinputを簡単に指定することを可能にします。

  • #{$all-button-inputs}
  • #{$all-button-inputs-hover}— all-button-inputs を指定するセレクターに :hover 疑似クラスを付与して出力します
  • #{$all-button-inputs-focus}— all-button-inputs を指定するセレクターに :focus 疑似クラスを付与して出力します
  • #{$all-button-inputs-active}— all-button-inputs を指定するセレクターに :active 疑似クラスを付与して出力します

注: 変数を渡すためには #{…} のような補完コードを使う必要があります。

{% highlight scss %} #{$all-text-inputs} { border: 1px solid green; } #{$all-text-inputs-hover} { // :hover 疑似クラスを対象とする background: yellow; } #{$all-text-inputs-focus} { // :focus 疑似クラスを対象とする background: white; } #{$all-text-inputs-active} { // Target the :active pseudo-class background: blue; } #{$all-button-inputs} { background: green; } #{$all-button-inputs-hover} { // :hover 疑似クラスを対象とする background: yellow; } #{$all-button-inputs-focus} { // :focus 疑似クラスを対象とする background: white; } #{$all-button-inputs-active} { // :active 疑似クラスを対象とする background: blue; } {% endhighlight %}
