Position

ソースを見る

要素の位置を設定する省略記法です。

第1引数は省略可能で、デフォルトは relative です。第2引数は、スペース区切りのリストでtop 、 right 、 bottom 、 left の値を指定します。これは、スタンダードなCSSの省略記法に準拠します。

注) null 値は無視されます。つまり、次の例では、right と bottom プロパティの宣言は生成されないということです。

{% highlight scss %} .element { @include position(relative, 0 null null 10em); } {% endhighlight %}

CSS Output

{% highlight css %} .element { position: relative; top: 0; left: 10em; } {% endhighlight %}
