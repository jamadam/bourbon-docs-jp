Hide Text

ソースを見る

背景画像を表示するためにテキストを隠します(例えばロゴなど)。これは the "Kellum Method" image replacement をベースとしています。

この画像置換は、HTML5 Boilerplate method を改善したもので、同様の利点があります。

高さの宣言はもはや必要なくなりましたが、インラインレベル要素にはブロックレベルの display スタイル("block", "inine-block", 等)を適用する必要があります。

{% highlight scss %} .element { @include hide-text; background-image: url(logo.png); } {% endhighlight %}

CSS出力

{% highlight css %} .element { text-indent: 101%; overflow: hidden; white-space: nowrap; background-image: url(logo.png); } {% endhighlight %}
