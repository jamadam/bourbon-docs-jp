Hide Text

ソースを見る

hide-text は、画像の置き換えのミックスインです。これは HTML5 Boilerplate image replacement をベースとしています。

この画像置き換えの手法は、 text-indent の手法の代替で、これには パフォーマンとその他の利点 があります。

この動作のためには、height 宣言が必須で(REQUIRED)、引数として指定可能です(デフォルトは 1em です)。

{% highlight scss %} .element { @include hide-text; background-image: url(logo.png); } .element { @include hide-text(2em); // passing an optional height argument } {% endhighlight %}

CSS Output

{% highlight css %} .element { height: 1em; line-height: 1.5; overflow: hidden; background-image: url(logo.png); } .element::before { content: ""; display: block; width: 0; height: 100%; } {% endhighlight %}
