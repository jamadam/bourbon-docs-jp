Golden Ratio

Deprecation 警告: golden-ratio 関数は廃止予定となり、次のメジャーバージョンのリリースから削除されます。代わりに modular-scale を使用してください。

与えられた値の黄金比を返します。第1引数にピクセル値またはem値を与えてください。また、 ...-3, -2, -1, 0, 1, 2, 3... のような増分の整数値を引数にとります(REQUIRED)。

注: golden-ratio 関数は、Sassの abs() や floor() 、 ceil() の関数で包むことができ、それぞれ、絶対値、切り捨て、切り上げを行います。

{% highlight scss %} // 正数は黄金比を加算する font-size: golden-ratio(14px, 1); // returns: 22.652px // 負数は黄金比を減算する font-size: golden-ratio(14px, -1); // returns: 8.653px {% endhighlight %}
