Modular Scale

ソースを見る

この関数は、定義されたスケールで上昇または下降し、調整済みの値を返します。これは、あなたのプロジェクトを通じて一貫した測量法と空間的相関関係を確立するのに役立ちます。私たちは 定義済み変数 として、よく使われるスケールのリストを提供しています。

{% highlight scss %} div { // デフォルトスケールで2ステップ上昇する font-size: modular-scale(2); // 結果は 1.77689em // デフォルトスケールで1ステップ下降する font-size: modular-scale(-1); // 結果は 0.75019em // 2emを基点として、デフォルトスケールで3ステップ上昇する font-size: modular-scale(3, 2em); // 結果は 4.73719em } {% endhighlight %}

デフォルトのスケールは完全4度 (1.333) で、グローバルにオーバーライドすることができます。

{% highlight scss %} $modular-scale-ratio: $golden; $modular-scale-base: 1.2em; div { font-size: modular-scale(2); // 結果は 3.14151em } div { font-size: modular-scale(3, 2em, 1.234); // 結果は 3.75816em } {% endhighlight %}

さらに、この関数は、2つの基準値を渡すことで二本鎖のスケールもサポートします。

{% highlight scss %} div { font-size: modular-scale(3, 1em 1.6em, $major-seventh); // 結果は 3em } {% endhighlight %}

注: この関数は、sassの abs() や floor() 、 ceil() の関数で包むことができ、それぞれ、絶対値、切り捨て、切り上げを行います。
