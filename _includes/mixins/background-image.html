Background Image

ソースを見る スペックを見る

background-image ミックスインは、背景画像や linear/radial-gradients を一つのbackground-image プロパティにカンマ区切りで列記するのに便利です。 background-image ミックスインは最大で10の 背景画像 をサポートします。

linear-gradient 関数 と radial-gradient 関数 とを組み合わせて使用してください。

{% highlight scss %} // 画像資源と一つの linear-gradient @include background-image(url("/images/a.png"), linear-gradient(white 0, yellow 50%, transparent 50%)); // 複数の linear-gradients の例 @include background-image(linear-gradient(hsla(0, 100%, 100%, 0.25) 0%, hsla(0, 100%, 100%, 0.08) 50%, transparent 50%), linear-gradient(#4e7ba3, darken(#4e7ba4, 10%))); // サポートしていない @include background-image(url("/images/a.png") center no-repeat, url("images/b.png") left repeat); // Background-image は短縮プロパティではないため、 これは意味をなさない。 {% endhighlight %}

デモ

短縮記法について

すべての CSS の background プロパティはカンマ区切りの値をサポートします。複数の背景画像のために、各画像に position、 repeat などの background プロパティを指定できます。例えば、

{% highlight scss %} @include background-image(url("/images/a.png"), url("images/b.png")); background-position: center top, center; background-repeat: no-repeat, repeat-x; {% endhighlight %}
