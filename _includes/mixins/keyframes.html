Keyframes

ソースを見る スペックを見る

クリーンなベンダープレフィックス付きのキーフレームを生成するミックスインです。

{% highlight scss %} @include keyframes(fadeIn) { from { @include transform(scale(0)); } to { @include transform(scale(1)); } } {% endhighlight %}

CSS出力

{% highlight scss %} @-webkit-keyframes fadeIn { from { -webkit-transform: scale(0); } to { -webkit-transform: scale(1); } } @-moz-keyframes fadeIn { from { -moz-transform: scale(0); } to { -moz-transform: scale(1); } } @-o-keyframes fadeIn { from { -o-transform: scale(0); } to { -o-transform: scale(1); } } @keyframes fadeIn { from { transform: scale(0); } to { transform: scale(1); } } {% endhighlight %}
