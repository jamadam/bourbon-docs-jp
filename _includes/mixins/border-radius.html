Border Radius (短縮形)

ソースを見る

これらのミックスインは、ボックスの指定の辺の両端角に角丸を追加する短縮形シンタックスを提供します。

{% highlight scss %} @include border-top-radius(5px); @include border-right-radius(5px); @include border-bottom-radius(5px); @include border-left-radius(5px); {% endhighlight %}

例

{% highlight scss %} .element { @include border-top-radius(5px); } {% endhighlight %}

CSS出力

{% highlight scss %} .element { border-top-left-radius: 5px; border-top-right-radius: 5px; } {% endhighlight %}
