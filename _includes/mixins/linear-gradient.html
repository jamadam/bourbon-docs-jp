Linear Gradient

ソースを見る スペックを見る

グラデーションポジションはオプションです。ポジションは角度で指定できます (90deg) 。ミックスインでは最大10ヶのカラーストップをサポートします。

このミックスインは background-color: #first-color; の宣言をフォールバックとして出力します。$fallback 変数を渡すことで、フォールバックの色を変更できます。

{% highlight scss %} @include linear-gradient(#1e5799, #2989d8); @include linear-gradient(to top, #8fdce5, #3dc3d1); @include linear-gradient(to top, #8fdce5, #3dc3d1, $fallback: red); @include linear-gradient(50deg, #1e5799 0%, #2989d8 50%, #207cca 51%, #7db9e8 100%); {% endhighlight %}
